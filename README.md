# Day 2 Practice - CSV Mini Data Pipeline

This project is a small practice exercise where I worked with real student data from a CSV file. The goal was to read the data, check for problems, clean it, and create useful output files that are easier to understand.

## What this project does

The script reads the raw student dataset, checks for common issues such as missing values, invalid numbers, and inconsistent text, then cleans the data and saves the results in a more structured format.

## Input file

- week-1/day-2-practice/data/students_raw.csv

## Output files

- week-1/day-2-practice/output/students_clean.csv
- week-1/day-2-practice/output/data_quality_report.txt
- week-1/day-2-practice/output/summary_report.txt

## How to run it

Open the terminal and go to the practice folder, then run:

```bash
python csv_pipeline.py
```

## What I practiced today

- working with CSV files
- using lists and dictionaries
- using loops and conditions
- cleaning messy data
- writing simple functions
- creating reports in the terminal and as text files
- organizing a small project in GitHub

This was a useful exercise for learning how real data workflows usually start: raw data comes in, it needs to be checked, cleaned, and turned into something usable.
